# MyArrayProject

## Опис
Цей проєкт містить програму на C++, яка працює з масивом цілих чисел.  
Програма виконує дві функції:  
1. **f9** — знаходить індекс найменшого додатного елемента масиву.  
2. **f16** — видаляє всі нулі з масиву.  

Після введення користувачем масиву, програма демонструє обидві функції та виводить результати.

## Використані технології
- Мова програмування: C++
- Стандартна бібліотека: iostream, iomanip

## Як запустити програму
1. Скопіюйте файл `main.cpp` на свій комп’ютер.  
2. Відкрийте термінал і скомпілюйте програму командою:
   ```bash
   g++ main.cpp -o MyArrayProject
